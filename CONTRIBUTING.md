# Contributing to CIPHER

First off, thank you for considering contributing to CIPHER!

## Getting Started

1. Fork the repository and create your branch from `main`.
2. Ensure you have the required version of Go installed.
3. Install dependencies using `go mod download`.

## Development Process

1. If you've added code that should be tested, add tests.
2. Ensure the test suite passes (`make test`).
3. Ensure your code is formatted properly (`gofmt -w .`).
4. Issue that pull request!

## Pull Request Guidelines

- Fill out the PR template completely.
- Keep PRs focused on a single feature or bug fix.
- Ensure CI checks pass before asking for a review.
